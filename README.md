

# 🧮 Console-Based Calculator

A beginner-friendly **Python console calculator** that performs basic arithmetic operations using a simple menu-driven interface.

This project is ideal for learning Python fundamentals such as functions, user input, conditional statements, and loops.

---

## 📌 Project Overview

The **Console-Based Calculator** allows users to perform calculations directly from the terminal.

It supports multiple operations and keeps running until the user chooses to exit.

---

## ✨ Features

* Addition
* Subtraction
* Multiplication
* Division
* Clear and simple menu
* Handles invalid choices
* Runs continuously until exit

---

## 🛠️ Technologies Used

* Python 3
* Command-line interface

## ▶️ How to Run

1. Clone the repository

```
git clone <your-repository-url>
```

2. Open the project folder

```
cd console_calculator
```

3. Run the program

```
python main.py
```

---

## 🧭 How It Works

After running the program, a menu appears:

* Add
* Subtract
* Multiply
* Divide
* Exit

The user selects an operation, enters two numbers, and the result is displayed on the screen.
